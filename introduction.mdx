The Greenhouse Monitoring System is a real-time data collection and monitoring solution designed to track environmental
conditions in a greenhouse. The system consists of:
- Raspberry Pi as the central processing unit
- Various sensors for environmental monitoring
- FastAPI backend for data collection and API exposure
- MQTT for real-time data transmission
- Database for historical data storage

You are in the right place to learn how to set up and use the system.
